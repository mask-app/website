mask Privacy Policy

Effective date: 2026-03-06

1. Data Processing Principle

mask is local-first. OCR and core visual understanding run on your device by default, and raw images are not uploaded.

2. Data That May Be Sent

If you manually choose a cloud translation provider, required text requests are sent to the third-party API you configured. You can switch to local mode anytime.

3. Credential Storage

API keys should be stored in secure system storage (for example Keychain), not plain-text files.

4. Contact

For privacy questions: hello@mask-app.com
