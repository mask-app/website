隐私政策

生效日期：2026-03-18 ｜ 适用网站与客户端产品：Mask

本隐私政策（以下简称“本政策”）构成你与 Mask 产品运营方之间就信息处理事项达成的说明性文件。你访问、下载、安装或继续使用 Mask 网站及客户端产品（合称“本服务”）的，即视为你已阅读、理解并同意受本政策约束。

1. 定义与适用范围

除非上下文另有说明，本政策中的“我们”指 Mask 产品运营方；“你”或“用户”指使用本服务的自然人或实体；“第三方模型服务商”指由你自行选择并配置的模型 API 服务提供方（包括但不限于 OpenAI、Google Gemini、DeepSeek、Xiaomi Mimo、Ollama 远程部署等）。

2. 信息处理范围（本地优先）

本服务采用“本地优先”架构。在默认配置下，以下信息主要保存在你的设备本地：

  • 你手动配置的 API Key（通过本地安全存储机制保存）；
  • 对话历史、翻译记录、设置项等本地使用数据；
  • 运行日志与错误信息（用于本地故障排查）。

3. 我们不保留的内容

在通常业务流程下，Mask 服务端不持久化保存你的 API Key、对话历史、原始截图或翻译文本内容。除你主动发起并向外部服务发送请求外，本服务不会将前述内容上传至 Mask 服务端进行中心化历史归档。

4. 第三方模型服务与责任边界

当你启用第三方模型能力时，相关请求将按照你的选择发送至对应第三方模型服务商。

  • 第三方模型服务商独立决定其数据处理规则、日志保留周期及合规义务履行方式；
  • 各模型服务的隐私政策、服务条款及后续更新由该等服务商单独发布并解释；
  • 我们无法控制第三方模型服务商对你提交数据的存储、分析或其他进一步处理行为。

重要提示：你应在使用前自行审阅并接受相应第三方模型服务商的隐私政策与服务条款。在适用法律允许范围内，我们对第三方模型服务商的数据处理行为不承担责任。

5. 信息处理目的

我们仅在实现产品功能、运行稳定性与安全保障所必需的范围内处理信息，主要包括：

  • 执行 OCR、翻译与上下文处理；
  • 保存你的本地偏好配置与会话状态；
  • 定位客户端错误并改进产品体验。

6. 数据存储、保留与删除

  • 本地数据由你自行控制，你可随时在设备侧删除、重置或覆盖；
  • 当你卸载应用或清理本地数据后，相关信息将按你的系统行为从设备移除；
  • 我们不建立用户云端数据档案，亦不提供中心化历史数据托管。

7. 数据安全措施

我们采取与本服务规模和风险水平相匹配的合理安全措施，包括本地安全存储机制、最小化上传策略及必要权限控制。但你理解并同意，任何技术与管理措施均无法保证信息系统达到绝对安全。

8. 你的权利与选择

  • 你有权在本地查看、修改或删除已保存的配置及历史记录；
  • 你可随时更换、停用或撤销第三方 API Key；
  • 你可停止使用本服务并删除本地数据。

9. 未成年人条款

本服务不面向未满 13 周岁（或你所在法域规定的更高最低法定年龄）的未成年人提供定向服务。若你作为监护人认为存在不当信息处理情形，请通过本政策载明方式与我们联系。

10. 政策更新与生效

我们有权基于产品功能调整、业务变化或合规要求对本政策进行更新。更新版本将通过网站或应用内方式发布；涉及重大变更的，我们将提供合理且显著的提示。除另有说明外，更新后的政策自发布之日起生效。

11. 联系我们

如你对本政策或相关信息处理事项有任何疑问、意见或投诉，可通过以下方式联系我们： hello@mask-app.com
