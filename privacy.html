Mask 隐私政策

生效日期：2026-03-18 ｜ 适用网站与客户端产品：Mask

本隐私政策说明 Mask（以下简称“本应用”）如何收集、使用、存储与保护你的信息。你使用本应用，即表示你已阅读并理解本政策。

1. 我们处理的信息范围

本应用采用“本地优先”设计。默认情况下，以下信息会保存在你的设备本地：

  • 你手动配置的 API Key（本地安全存储）
  • 对话历史、翻译记录、设置项等本地使用数据
  • 运行日志与错误信息（用于本地排查）

2. 我们不会保留的内容

Mask 服务端不保存你的 API Key、对话历史、原始截图或翻译文本内容。除你主动发起网络请求外，本应用不会将前述数据上传到 Mask 服务端持久化存储。

3. 第三方模型服务与责任边界

当你使用 OpenAI、Google Gemini、DeepSeek、Xiaomi Mimo、Ollama（远程部署）等模型能力时，相关请求将发送至你所选择或配置的第三方服务商。

  • 第三方服务商对其数据处理规则、日志保留策略和合规义务负责。
  • 各模型服务的隐私政策由其服务商单独制定并更新。
  • 本应用无法控制第三方服务商如何存储、分析或二次处理你发送的数据。

重要说明：你应在使用前自行阅读并接受相应模型服务商的隐私政策与服务条款。本应用不对第三方模型服务商的数据处理行为承担责任。

4. 数据用途

我们仅在实现产品功能与稳定性所必需的范围内处理信息，主要用于：

  • 执行 OCR、翻译与上下文处理
  • 保存你的本地偏好配置与会话状态
  • 定位客户端错误并改进产品体验

5. 数据存储与保留

  • 本地数据由你自行控制，可随时删除或重置。
  • 你卸载应用或清理本地数据后，相关信息将从设备移除（以你的系统行为为准）。
  • 本应用不建立用户云端数据档案，不提供中心化历史数据托管。

6. 数据安全

我们采取合理安全措施保护数据，包括本地安全存储机制、最小化上传策略与权限控制。但请理解，任何系统都无法保证绝对安全。

7. 你的权利与选择

  • 你可以在本地查看、修改或删除已保存的配置和历史记录。
  • 你可以更换、停用或撤销第三方 API Key。
  • 你可以停止使用本应用并删除本地数据。

8. 未成年人

本应用不面向未满 13 周岁（或你所在法域最低法定年龄）的未成年人提供定向服务。若你是监护人并发现相关信息处理问题，请联系我们。

9. 政策更新

我们可能根据产品与合规要求更新本政策。更新后将通过网站或应用内方式公告，重大变更会给出显著提示。

10. 联系我们

隐私相关问题可通过以下方式联系： hello@mask-app.com
